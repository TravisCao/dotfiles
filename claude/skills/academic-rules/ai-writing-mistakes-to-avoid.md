# AI Writing Patterns to Avoid in Academic Papers

Based on our discussion and [Wikipedia: Signs of AI Writing](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing)

---

## 1. Words to NEVER Use

### Overused AI Vocabulary
| Word | Why Avoid |
|------|-----------|
| delve | Classic AI tell |
| crucial, key, valuable | Overused emphasis |
| enduring, enhance, fostering | Vague positivity |
| garner, vibrant | Promotional tone |
| intricate, intricacies | Pretentious |
| tapestry (abstract) | "rich tapestry of..." |
| interplay | Vague |
| testament | "stands as a testament" |
| underscore (verb) | Use "emphasize" sparingly instead |
| showcase | Promotional |
| pivotal | Overused |
| landscape (abstract) | "evolving landscape" |

### Non-Academic Words Claude Uses
| Avoid | Use Instead |
|-------|-------------|
| stem from | result from, arise from, are caused by |
| circumvent | avoid, bypass |
| delve into | examine, investigate, explore |
| facilitate | enable, allow |
| leverage | use, apply |
| utilize | use |

### Promotional Words
| Word/Phrase | Problem |
|-------------|---------|
| nestled | "nestled in the heart of" |
| in the heart of | Cliche |
| boasts | Promotional |
| renowned | Often unverifiable |
| rich (figurative) | "rich cultural heritage" |
| profound | Overstatement |
| groundbreaking | Overclaiming |

## 2. Sentence Patterns to Avoid

### Overemphasizing Significance
```
BAD:  "stands as a testament to..."
BAD:  "plays a vital/crucial/pivotal role in..."
BAD:  "underscores its importance/significance..."
BAD:  "reflects broader trends in..."
BAD:  "marks a significant shift in..."
GOOD: Be specific about what it actually does
```

### Superficial "-ing" Analyses
```
BAD:  "The method improves accuracy, highlighting its potential for..."
BAD:  "This approach reduces costs, demonstrating the effectiveness of..."
GOOD: Make concrete claims with evidence
```

### Negative Parallelisms
```
BAD:  "not only X but also Y"
BAD:  "it's not just about X, it's about Y"
BAD:  "X is not A, but rather B"
GOOD: Direct statements without rhetorical contrast
```

### Replacing Simple "is/are"
```
BAD:  "serves as a", "stands as a", "represents a"
BAD:  "boasts", "features", "offers"
GOOD: Use simple "is" or "are"
```

### Section/Paragraph Summaries
```
BAD:  "In summary, ..."
BAD:  "Overall, ..."
BAD:  "In conclusion, ..." (except in actual conclusion)
GOOD: Let the content speak for itself
```

### Rule of Three
```
BAD:  "fast, efficient, and reliable"
BAD:  "planning, implementing, and evaluating"
BAD:  "reduces costs, improves efficiency, and enhances reliability"
GOOD: Use only adjectives/phrases that add real meaning
```

### Vague Attributions
```
BAD:  "Observers have noted..."
BAD:  "Experts argue..."
BAD:  "Industry reports suggest..."
BAD:  "Studies have shown..."
GOOD: Cite specific sources with names
```

## 3. Formatting Issues

### Overuse of Boldface
- NO: Bold "key takeaways" at end of sections
- NO: Bold every important term
- YES: Bold only for defined terms on first use (if journal requires)

### Overuse of Lists
- NO: Converting prose into bullet points unnecessarily
- NO: Lists for things that should be sentences
- YES: Lists only for genuinely parallel items (steps, requirements)

### Overuse of Colons
```
BAD:  "The method has three advantages: speed, accuracy, and simplicity."
BAD:  "We propose: a new framework for..."
GOOD: Use colons sparingly, often a period works better
```

### Questions in Academic Writing
```
BAD:  "What makes this approach different?"
BAD:  "How can we solve this problem?"
GOOD: Use declarative statements
```

### Curly Quotation Marks
```
BAD:  "quoted text" (curly quotes)
BAD:  'apostrophe' (curly apostrophe)
GOOD: "quoted text" (straight quotes)
GOOD: 'apostrophe' (straight apostrophe)
```

### Title Case in Headings
```
BAD:  "Results And Discussion"
GOOD: "Results and discussion" (sentence case, per IEEE)
```

## 4. Tone Problems

### Promotional Language
```
BAD:  "This groundbreaking approach revolutionizes..."
BAD:  "Our novel method achieves unprecedented..."
GOOD: "This method reduces computation time by 40%"
GOOD: "Unlike [specific prior work], our method preserves..."
```

### Hedging (use sparingly)
```
CAUTION: may, might, could, perhaps, possibly
- OK for genuine uncertainty
- BAD when overused to weaken claims that have evidence
```

### Excessive "Additionally"
```
BAD:  Starting multiple sentences/paragraphs with "Additionally"
GOOD: Vary transitions or omit if connection is clear
```

## 5. Quick Checklist

Before submitting, verify NO:
- [ ] Overused AI words (delve, crucial, pivotal, tapestry, etc.)
- [ ] Non-academic words (stem from, circumvent, leverage)
- [ ] "serves as" / "stands as" instead of "is"
- [ ] "-ing" superficial analyses at sentence ends
- [ ] "not only...but also" constructions
- [ ] Paragraph summaries ("In summary", "Overall")
- [ ] Rule of three patterns
- [ ] Vague attributions ("Experts argue")
- [ ] Excessive boldface or lists
- [ ] Questions in prose
- [ ] Curly quotes
- [ ] Promotional language
- [ ] Excessive colons
- [ ] Multiple "Additionally" sentence starters
